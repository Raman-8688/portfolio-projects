# Java Full Stack Portfolio Redesign Plan (Updated)

We will overhaul the existing portfolio to achieve a top-tier, enterprise-level design. It will showcase a 2-year experienced Java Full Stack Developer specializing in Spring Boot, Angular, Microservices, and cloud deployments.

## User Review Required

> [!IMPORTANT]
> **Refined Design Decisions:**
> 1. **Default Theme Strategy:** Dark mode will be the default theme instead of completely forcing dark-only mode. This preserves the theme flexibility structure.
> 2. **Global Utility Classes & Tokens:** We will define layout, cards, buttons, and text utilities globally in `styles.css` using dynamic variables for spacing and styling.
> 3. **Engineering-Focused Projects:** Project cards will be modified to clearly communicate technical impact: problem solved, system topology, container orchestration (Docker/K8s), security protocols (JWT/RBAC), and deployment pipelines (CI/CD).
> 4. **No Stock Photo Backdrops:** Background image urls will be completely removed. We will use clean, performance-friendly CSS grid overlays, subtle radial glow meshes, and high-quality responsive CSS/SVG schematics.
